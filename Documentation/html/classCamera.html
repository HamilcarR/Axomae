Constructor for the Camera class , which initializes various properties of the camera.

Parameters
radiansThe field of view angle in radians for the camera.
screenA pointer to an object of type ScreenSize, which contains information about the size of the screen or window where the camera will be rendering.
clip_nearThe distance from the camera to the near clipping plane. Any objects closer to the camera than this distance will not be visible.
clip_farThe far plane of the camera's frustum, which determines the maximum distance at which objects will be visible.
pointerThe "pointer" parameter is a pointer to a MouseState object, which is used to track the state of the mouse (e.g. position, button clicks) for camera movement and control.
