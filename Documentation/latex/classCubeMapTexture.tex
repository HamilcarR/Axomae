Initialize cubemap data. 

width $\ast$ height is the size of one single face. The total size of the cubemap will be \+: \begin{DoxyVerb}6 x width x height x sizeof(uint32_t) bytes 
\end{DoxyVerb}
 with height = width . Here is the layout for mapping the texture \+: \begin{DoxyVerb}      width² = RIGHT => GL_TEXTURE_CUBE_MAP_POSITIVE_X
  2 x width² = LEFT => GL_TEXTURE_CUBE_MAP_NEGATIVE_X
  3 x width² = TOP => GL_TEXTURE_CUBE_MAP_POSITIVE_Y
  4 x width² = BOTTOM => GL_TEXTURE_CUBE_MAP_NEGATIVE_Y
  5 x width² = BACK => GL_TEXTURE_CUBE_MAP_POSITIVE_Z
  6 x width² = FRONT => GL_TEXTURE_CUBE_MAP_NEGATIVE_Z
\end{DoxyVerb}
